# Assignment 6: Mobile Application Security (Static Analysis)

This repository contains the static analysis reports of Android and iOS applications performed using **MobSF (Mobile Security Framework)**.

## Applications Tested
- Android: `allsafe.apk`, `insecurebankv2.apk`, `dvba_v1.1.0.apk`
- iOS: `iGoat`

## Contents
- PDF reports for each application (`reports/` folder)
- Hash values and file info
- This README.md file

## Overview
The assignment focuses on identifying security issues like:
- Hardcoded secrets
- Debuggable APKs
- Insecure storage
- Exported activities
- App Transport Security issues (iOS)

All findings and recommendations are documented in the respective PDF reports.
