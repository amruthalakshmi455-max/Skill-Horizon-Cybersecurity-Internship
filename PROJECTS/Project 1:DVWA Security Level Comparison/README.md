# Project 1: DVWA Security Level Comparison

## Overview
This project analyzes how DVWA behaves across different security levels (Low, Medium, High, Impossible).

## Vulnerabilities Tested
- SQL Injection
- File Upload

## Environment
- OS: Kali Linux
- Web Application: DVWA
- Security Levels: Low, Medium, High, Impossible

## Key Learning
- How input filtering alone is insufficient
- Why parameterized queries are critical
- Importance of secure file handling
